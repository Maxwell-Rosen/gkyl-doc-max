dataset

Command line usage
^^^^^^^^^^^^^^^^^^

.. code-block:: bash

   $ pgkyl dataset --help
   Usage: pgkyl dataset [OPTIONS]

     Select data sets(s)

   Options:
     -i, --idx TEXT  Data set indices
     -a, --allsets   All data sets
     --help          Show this message and exit.
